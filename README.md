# ao-code-jam-2024


# Description

使用卡牌攻击敌人并防御自己。

每次抽取一张卡，可以获得攻击点数或防御点数。然而，如果你手中的点数总和达到或超过13点，你将失去所有卡牌。

卡牌点数从1到11不等。第13张卡是“死亡卡”，它会让你失去所有卡牌。而第12张卡是“四叶草卡”，它可以保护你免受“死亡卡”的影响。

游戏开始时，你将获得一张“四叶草卡”。

祝你好运，尽可能多地用“绿色炸弹”获胜！
